About

Trojsten Graph is the latest in the history of various graphs that documented the dating scene in Trojsten. What originated as a static page with rendered graph image of KSP people has evolved into...well, you can see for yourself.

This project is obviously not official, and is in no way supported by Trojsten o.z. The source code is open, with Github repository here
